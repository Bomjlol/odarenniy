# LED Controller v1.0

Этот проект представляет собой графический интерфейс для управления яркостью и временем работы двух светодиодов, подключенных к Arduino, с помощью Python и библиотеки Tkinter.

---

## Особенности

- Управление двумя светодиодами (Blue и Red)
- Установка времени задержки перед включением светодиода
- Регулировка яркости (0-100%)
- Простая и удобная графическая оболочка
- Работает через последовательное соединение с Arduino по COM-порту

---

## Требования

- Python 3.x
- Библиотеки:
  - `tkinter` (обычно входит в стандартный пакет Python)
  - `pyfirmata` (устанавливается через pip)
- Arduino с подключенными PWM-выходами (например, пины 3 и 5)

---

## Установка

1. Установите необходимые библиотеки:

```bash
pip install pyfirmata
Подключите Arduino к компьютеру и вставьте необходимую программу (если требуется).

Укажите корректный COM-порт в коде ("COM3" или другой ваш порт).

Использование
Запустите файл led_controller.py.
Введите время задержки в секундах.
Регулируйте яркость с помощью ползунка.
Нажимайте кнопку BLUE LED или RED LED для включения соответствующего светодиода на заданное время.
Для получения информации нажмите Справка.
Для выхода — Закрыть.
Внимание
Перед запуском убедитесь, что Arduino подключен и работает на выбранном COM-порту.
Этот проект использует PWM для управления яркостью светодиодов. Убедитесь, что ваши пины поддерживают PWM.
