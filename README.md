# Vault-Tec Power Interface

Это симулятор интерфейса управления электропитанием от Vault-Tec, выполненный в стиле Fallout с современным дизайном. Визуализируется отключение и включение питания, имитируется сканлайн, и есть звуковые эффекты для создания атмосферы ретрофутуризма.

---

## Основные возможности

- Включение и выключение питания кнопкой
- Визуальный индикатор и анимация сканлайна
- Эффект мерцания CRT-экрана
- Интерактивная кнопка "About"
- Минималистичный и современный дизайн в тёмных тонах
- Поддержка звуковых эффектов при включении/выключении (требуется установленная команда `afplay`, поддерживающая macOS; для Windows можете заменить, например, `playsound`)

---

## Требования

- Python 3.x
- Модули: `tkinter`, `PIL` (Pillow)
- Для воспроизведения звуков на macOS установлен `afplay`. Для Windows или Linux требуется заменить команду проигрывания, например, на `playsound`
- Изображение `vaultboy.png` должно находиться в папке с скриптом, или измените путь

---

## Установка

1. Убедитесь, что Python 3 установлен.
2. Установите Pillow, если ещё не установлен:
```bash
pip install pillow
Поместите изображение vaultboy.png в ту же папку, что и скрипт (или укажите полный путь).
Использование
Запустите скрипт:

bash

python your_script_name.py
Интерфейс откроется в окне.

Нажмите кнопку "OFF" для включения питания.
При включении появится сканлайн и будет изменяться яркость.
При выключении интерфейс отключится, и все анимации остановятся.
Нажмите "ABOUT" для информации.
